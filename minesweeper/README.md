# Minesweeper

This is a minesweeper game made in vanilla HTML, CSS, JavaScript

## Game Instruction

OBJECTIVE: All non-mine cells must be opened without opening a mine (36 mines in total)

CONTROLS: LEFT CLICK - open cell | RIGHT CLICK - set flag
